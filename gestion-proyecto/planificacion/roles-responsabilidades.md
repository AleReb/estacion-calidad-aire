# Roles y responsabilidades
Basado en el brochure técnico y las definiciones entregadas por el equipo.

| Rol | Responsable | Responsabilidades principales |
| --- | --- | --- |
| Director/a del proyecto | Zoe Fleming | Define lineamientos estratégicos, asegura recursos y alinea el proyecto con objetivos institucionales. |
| Coordinador del equipo | Luis Gómez | Coordina disciplinas, consolida reportes de avance y mantiene comunicación con interesados clave. |
| Coordinador de firmware y comunicaciones | Alejandro Rebolledo | Lidera programación del firmware, integra sensores con la plataforma y mantiene las telecomunicaciones GSM/4G y RS485/Modbus. Coordina electronica. |
| Coordinación de terreno | Pablo Ortiz | Gestiona despliegues de torres (2–5 m), logística de instalaciones y soporte de campo durante pruebas. |
| Coordinación de mediciones | Catalina Azola | Define rutinas de adquisición, asegura la altura de muestreo (1,5 m) y coordina muestreo pasivo. |
| Encargado de documentación regulatoria, QA/QC de datos y bases de datos | Christopher Silva | Mantiene documentación regulatoria, valida la calidad de los datos, gestiona respaldos y define controles sobre las bases almacenadas en `datos/`. |
| Administradora de datos de mediciones | Catalina Azola | Administra la carga de datos, verifica consistencia en `datos/datos-muestra` y coordina la entrega de datasets para análisis. |
| Diseñador de hardware y componentes | Rodrigo Ortega | Diseña el soporte físico, integra panel solar + banco de baterías y selecciona componentes mecánicos y de protección. |
| Diseñador electrónico | Sebastián Adonay | Desarrolla el diseño electrónico de adquisición, alimentación y acondicionamiento para sensores electroquímicos, ópticos y meteorológicos. |
