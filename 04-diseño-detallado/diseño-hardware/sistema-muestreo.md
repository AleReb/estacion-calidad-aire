# Sistema de muestreo
- Altura de captación de 1,5 m sobre nivel del suelo para alinearse con la exposición definida.
- Colectores para material particulado sedimentable compatibles con Resolución Exenta N° 1449 (MMA, 2023).
- Sensores ópticos y electroquímicos ubicados en flujo controlado dentro del colector para mantener la precisión declarada (±1 µg/m³ para PM, ±1 ppbv para O₃/NO₂/NO).
