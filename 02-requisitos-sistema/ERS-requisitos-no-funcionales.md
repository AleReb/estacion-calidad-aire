# ERS – Requisitos no funcionales
Valores tomados del brochure técnico.

| Categoría | Requisito | Fuente |
| --- | --- | --- |
| Medición de PM | El sensor Plantower PMS5003ST debe ofrecer límite de detección de 1 µg/m³ y precisión ±1 µg/m³ para PM₁.₀, PM₂.₅ y PM₁₀ mediante tecnología óptica láser. | Sección “Material particulado”. |
| Gases criterio | Los sensores Alphasense OX-B431, NO2-B43F, CO-B4 y NO-B4 deben cumplir límites de detección de 1 ppb (1 ppm para CO) con precisión ±1 ppbv (±1 ppmv para CO). | Sección “Gases criterio”. |
| Gases adicionales | El sensor CO₂ (Winsen MH-Z19B/MH-Z19C) debe entregar resolución de 1 ppm; los sensores SO₂ y COV totales tienen metas de 100 ppb y 1 ppm respectivamente. | Sección “Gases adicionales”. |
| Meteorológicos | La estación digital RS485 (Modbus) debe medir temperatura, humedad relativa, presión, velocidad y dirección del viento, además de variables opcionales (precipitación, radiación, luz, UV) tal como declara el fabricante ComWinTop. | Sección “Sensores meteorológicos” y tabla técnica. |
| Integración de datos | Los sensores deben estar sincronizados y operar en plataforma única para disponibilidad en tiempo casi real. | Descripción general. |
| Fiabilidad | Todos los sensores deben contar con especificaciones validadas y ser calibrados antes de entrar en operación. | Descripción general. |
