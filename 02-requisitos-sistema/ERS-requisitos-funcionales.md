# ERS – Requisitos funcionales
Los requisitos provienen directamente del brochure técnico.

1. El sistema debe integrar sensores electroquímicos, ópticos y meteorológicos dentro de la misma estructura vertical.
2. Debe recolectar, almacenar y transmitir datos en tiempo casi real.
3. Debe operar con una torre de 2–5 m que asegure exposición 360° y altura de medición a 1,5 m sobre el suelo.
4. Debe incorporar un módulo de comunicaciones GSM/4G y registrar los datos en una memoria microSD local.
5. Los sensores deben sincronizarse en una plataforma única para permitir análisis conjunto.
6. El hardware debe permitir acceso rápido a los módulos para mantenimiento y reemplazos.
7. Debe incluir sensores de material particulado (PMS5003ST), gases criterio (Alphasense serie B4/B43F), CO₂ (Winsen MH-Z19B/MH-Z19C), VOC (Winsen ZE40A-TVOC), SO₂ (Alphasense SO2-B4) y sensores meteorológicos RS485 (ComWinTop CWT-UWD-SDTHAPSIR, CWT-UR-S).
8. Debe soportar equipos de muestreo pasivo para material particulado sedimentable conforme a la Resolución Exenta N° 1449 (MMA, 2023).
9. Debe permitir aplicaciones de monitoreo ciudadano, validación de sensores de bajo costo, correlación con estaciones de referencia y evaluación de impactos urbanos/industriales.
