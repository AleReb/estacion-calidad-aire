# Vista de reportes en tiempo casi real
- El flujo de datos considera transmisión GSM/4G complementada con microSD para redundancia.
- Las aplicaciones objetivo (monitoreo ciudadano, validación de sensores, correlación con referencia y evaluación de impacto) se alimentan directamente de este flujo.
- La arquitectura autónoma (panel solar + baterías) permite mantener los reportes aun fuera de red eléctrica.
